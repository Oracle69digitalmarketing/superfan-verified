# Superfan Verified - Mobile Stub

## Quick start (requires Node.js & Expo)
\`\`\`
cd mobile
npm install
npm run start
\`\`\`

## Purpose
Displays a verified fan badge and leaderboard using the backend stub at http://localhost:4000.

## Flow
1. Fetch mock streaming history  
2. Verify via backend stub (/verify)  
3. Show badge and leaderboard
