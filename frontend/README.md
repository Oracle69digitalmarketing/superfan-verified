# Superfan Verified Web Stub

## Quick demo
Serves a minimal frontend that talks to the backend at http://localhost:4000

### Run
Open frontend/index.html in a browser. If running locally, you can serve it with:
\`\`\`
cd frontend
python -m http.server 8000
\`\`\`

Then visit: http://localhost:8000

## Features
- Verify fan activity (mock)  
- Display Superfan badge  
- Show top leaderboard for Taylor Swift
